# **Small Fishing Boat (Keelboat-sized)**

- **Size:** 30-50 ft long, re-inforced hull
    
- **Crew:** 5-6 people
    
- **Cargo Capacity:** ~1000–1500 lbs
    
- **Speed:** Rowing, ~5 mph (60 ft per round)
    

---

## 📊 Suggested Stats:

| Attribute              | Value                                |
| ---------------------- | ------------------------------------ |
| **Armor Class (AC)**   | 15 (wood)                            |
| **Hit Points (HP)**    | **100 HP** (reinforced hull)         |
| **Damage Threshold**   | 0 (small boats take full damage)     |
| **Immunities**         | Poison, Psychic, Non-magical disease |
| **Damage Resistances** | Piercing (arrows), unless critical   |

---

### Optional Rules:

- If you want realism, add:
    
    - **Sinking Threshold:** At 0 HP, boat sinks in **1d4 × 10 minutes** unless patched.
        
    - **Critical Hit:** Fire or siege weaponry does **double damage**.
        
    - **Crew Action:** Someone can **patch hull** with DC 12 Carpenter’s Tools (restore 1d10 HP per action).
